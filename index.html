How to use the resources:

Authoring Tool:

Follow the steps below to use the Authoring Tool:

  1. After downloading the tool, locate the downloaded files and unzip them.
  2. Within the unzipped files, locate the .exe file.
  3. Double-click the .exe file to start the program.
  4. If a warning message appears from Windows, please ignore it and proceed.
  5. Once the program has started, please wait for the presenter to guide you through how to use the software.

Moodle:

Follow these steps to enroll in your course on Moodle:

  1. Create your course using the Authoring Tool. Make sure to remember the name of your course.
  2. Click on the "Access Moodle" link above and sign in using your assigned student account.
  3. Once you're signed in, navigate to the "Home" tab.
  4. Find your course by pressing CMD + F (or CTRL + F on Windows) and typing the name of your course.
  5. Click on your course to enroll yourself.

3D Learning World:

Follow these steps to use the 3D Learning World:

  1. Ensure that your assigned student account is already enrolled in the Moodle course you created earlier.
  2. Click on the "Visit the 3D Learning World" link above to open the 3D Learning World.
  3. Log into the 3D Learning World using the credentials of your assigned student account.
  4. Now you can explore and play in the 3D Learning World!

Test Student Accounts:

We have created 50 test student accounts for you to use. These are named sequentially from 'testUser1' to 'testUser50', and each account's password is the same as its username. For example, 'testUser1' has the password 'testUser1'.

Please use the account that corresponds to the number of your table. So, if you're sitting at table 1, use 'testUser1'. If you're at table 2, use 'testUser2', and so on up to table 50.
